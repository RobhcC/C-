# C-
此存储库记录了基于C#和Winform开发的温湿度监控项目，以三层架构开发，支持Modbus RTU/TCP双协议解析、CRC校验、日志记录，适配PLC工控场景。

##  项目简介
- 支持串口COM + Socket网络双协议切换，兼容Modbus RTU/TCP标准报文
- 内置自动CRC16校验、报文格式化展示，一键解析读写寄存器数据
- 界面防重复点击、状态禁用、输入合法性校验，工业级稳定性设计
- 实时接收温湿度/设备寄存器数据，自动日志存档+数据可视化显示
- 轻量化架构，分层解耦（界面层/逻辑层/通信层），易扩展二次开发
- 数据可导出TXT/CSV，可同时批量读写多个寄存器
- 使用异步编程，减少了程序的阻塞与卡顿现象

- ##  运行环境
- 开发框架：.NET Framework 4.7.2
- 开发工具：Visual Studio 2022
- 依赖组件：System.IO.Ports（串口原生库，无第三方付费依赖）
- 适配设备：西门子/三菱PLC、Modbus从机设备
